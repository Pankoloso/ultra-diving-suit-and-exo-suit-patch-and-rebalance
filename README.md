This mod was created due to compatibility patches between individual mods were overriding one another.
Link to the steam page: https://steamcommunity.com/sharedfiles/filedetails/?id=3721248317

This mod combines all the different changes/resistances form following mods:

-Enhanced armaments - https://steamcommunity.com/sharedfiles/filedetails/?id=2764968387

-RRR armaments - https://steamcommunity.com/sharedfiles/filedetails/?id=3327924275

-NT thermal - https://steamcommunity.com/sharedfiles/filedetails/?id=3648890424

-Baroverhaul - https://steamcommunity.com/sharedfiles/filedetails/?id=3279470465

-Hazardus reactors - https://steamcommunity.com/workshop/filedetails/?id=2547888957

-Chemical agents - https://steamcommunity.com/sharedfiles/filedetails/?id=2974772300

All diving suits and exo suits get:
-Protection from chemical agents
-battery slot for hearter from NT thermal
-Protection from contamination

All hazmat suits and hazmat themed gear get:
-Protection from chemical agents

Anechoic diving suit gets:
-increased protection from the sonar to make an effective alternative and make room for 2 suits with lower sonar protection

RRR armament upgraded suits with accustic sensors get:
-added resistance to sonar damage

Chemical agents (nerfed compared to original mod since there were too op, might release this aprt to be available as a seperate mod):
-advanced hazmat suit - lost thermal vision but requires health scanner to craft instead
-advanced hazmat suit (armored) - lowered most combat oriented resistanced but kept decently high gunshot wound resistance, now requres thermal googles and durasteel to craft
-E.O.D. suit - slighlty lowered most combat oriented resistanced but mostly gunshot wound resistance, now requires durasteel to craft
